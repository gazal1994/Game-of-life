# Game of Life


## How to Play
1. Run the script `main.py`.
2. Click on cells to toggle them alive or dead.
3. Press the space bar to pause or resume the simulation.
4. Press 'c' to clear the grid.
5. Press 'g' to generate a random initial configuration.

## Controls
- Click: Toggle cell state
- Space bar: Pause/Resume
- 'c': Clear grid
- 'g': Generate random initial configuration

## Dependencies
- Python 3.x
- Pygame

## Usage
Run the main script:
```
python main.py [file_name] [max_iterations]
```
- `max_iterations` (optional): Maximum number of iterations before stopping (default is 1000).
- `file_name` (optional): input of points.
